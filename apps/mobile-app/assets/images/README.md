# Images Directory

Place all image assets here.

## Required Images

- `app_icon.png` - App icon (1024x1024)
- `app_icon_foreground.png` - App icon foreground (adaptive icon)
- `splash_logo.png` - Splash screen logo
- `logo.png` - Main logo
- `logo_dark.png` - Dark mode logo

## Optional Images

- `placeholder_camera.png` - Camera placeholder
- `placeholder_avatar.png` - User avatar placeholder
- `empty_state.png` - Empty state illustration

# Lottie Animations

هذا المجلد يحتوي على أنيميشن Lottie للتطبيق.

## الأنيميشن المطلوبة:

### 1. splash_animation.json
- أنيميشن شاشة البداية
- يمكن أن تكون كاميرا أو شعار متحرك
- مدة: 2-3 ثواني

### 2. loading.json
- أنيميشن التحميل
- دائري أو خطي

### 3. empty_state.json
- أنيميشن عند عدم وجود بيانات
- صندوق فارغ أو شاشة فارغة

### 4. error.json
- أنيميشن عند حدوث خطأ
- علامة تعجب أو خطأ

### 5. success.json
- أنيميشن عند النجاح
- علامة صح أو نجاح

## المصادر:
- LottieFiles: https://lottiefiles.com/
- يمكن البحث عن أنيميشن مجانية
- تأكد من الترخيص قبل الاستخدام

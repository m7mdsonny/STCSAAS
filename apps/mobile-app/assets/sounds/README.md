# Notification Sounds

هذا المجلد يحتوي على أصوات الإشعارات للتطبيق.

## الأصوات المطلوبة:

### 1. alert_critical.mp3
- صوت قوي وطويل للتنبيهات الحرجة
- مدة: 3-5 ثواني
- نوع: إنذار قوي

### 2. alert_high.mp3
- صوت متوسط القوة للتنبيهات العالية
- مدة: 2-3 ثواني
- نوع: تنبيه واضح

### 3. alert_medium.mp3
- صوت عادي للتنبيهات المتوسطة
- مدة: 1-2 ثانية
- نوع: إشعار عادي

### 4. alert_low.mp3
- صوت خفيف للتنبيهات المنخفضة
- مدة: 1 ثانية
- نوع: نغمة بسيطة

## ملاحظات:
- يجب أن تكون الملفات بصيغة MP3 أو WAV
- الحجم يجب أن يكون صغير (أقل من 100KB لكل ملف)
- جودة الصوت: 128kbps كافية

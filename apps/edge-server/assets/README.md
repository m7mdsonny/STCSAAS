# Assets Directory

Place the following files here for the Windows installer build:

## Required Files

### icon.ico
Application icon (256x256 recommended, multiple sizes embedded)
- Used for the executable
- Used for installer
- Used for Windows shortcuts

### wizard_large.bmp
Inno Setup wizard large image (164 x 314 pixels)
- Displayed on the left side of Welcome and Finish pages

### wizard_small.bmp
Inno Setup wizard small image (55 x 58 pixels)
- Displayed in the top-right corner of other pages

## Optional Files

### splash.png
Splash screen image (optional)

### banner.png
Setup banner image (optional)

## Creating Icons

You can use tools like:
- IcoFx (Windows)
- GIMP with ico plugin
- Online converters like convertio.co

The icon should include these sizes:
- 16x16
- 32x32
- 48x48
- 64x64
- 128x128
- 256x256
